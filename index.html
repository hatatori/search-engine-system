Busca HTML simples

1. Baixe o arquivo aqui.

2. Crie um input com o nome da class de "pesquisa".
<input class="pesquisa">

3. Crie uma lista com o nome de "lista", não precisa ser necessariamente um ul, pode ser uma div.
<ul class="lista"></ul>

4. Em cada elemento dentro da lista ponha um atributo chamado nome.
<li nome="coelho"> <img src="imgs/coelho.jpg"> Coelho </li>
<li nome="cavalo"> <img src="imgs/coelho.jpg"> Coelho </li>

5. Jogue toda estrutura criada acima para dentro de uma div.

6. Insira o js pra dentro.
<script src="sistema_de_busca.js"></script>

7. O arquivo final ficará mais ou menos assim.

<div>
  <input class="pesquisa">
  <ul>
    <li nome="coelho">Coelho</li>
    <li nome="cavalo">Cavalo</li>
  </ul>
</div>

<script src="sistema_de_busca.js"></script>
