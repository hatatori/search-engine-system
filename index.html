Busca HTML simples

1. Adicione o script

<script src="sistema_de_busca.js</script>
<script src="https://hatatori.github.io/search-engine-system/sistema_de_busca.js</script>


2. Crie um input com o nome da atributo search e ponha o valor da lista referente

<input search="animallist">


3. O nome do id da tag tem que o ser o mesmo que está contido dentro de search do input. Os elementos devem conter um atributo nome.

<div id="animallist">
<div nome="coelho"> Coelho </div>
<div nome="cavalo"> cavalo </div>
</div>


5. HTML da página abaixo ou Página de teste

<input search="animals">

<ul id="animals">
    <li nome="cachorro">Cachorro</li>
    <li nome="cavalo">Cavalo</li>
    <li nome="coelho">Coelho</li>
    <li nome="gato">Gato</li>
    <li nome="passaro">Pássaro</li>
    <li nome="peixe">Peixe</li>
    <li nome="tigre">Tigre</li>
</ul>
